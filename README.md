# Calculadora-simples
Este projeto consiste em uma calculadora simples, desenvolvida com o objetivo de realizar operações matemáticas básicas de forma rápida e intuitiva. É ideal para fins de aprendizado, treino de lógica e prática de programação.
 Funcionalidades:
A calculadora permite realizar as seguintes operações:
	•	➕ Adição
	•	➖ Subtração
	•	✖️ Multiplicação
	•	➗ Divisão
O usuário pode inserir dois valores numéricos e escolher a operação desejada, obtendo o resultado imediatamente.
Objetivo do Projeto
Este projeto foi criado para:
	•	Praticar conceitos básicos de programação
	•	Trabalhar com entrada e saída de dados
	•	Aplicar lógica condicional para seleção de operações
	•	Entender o funcionamento de uma aplicação simples e funcional.
	Tecnologias Utilizadas
	•	Linguagem de programação: (HTML, CSS e JavaScript )
	•	Estrutura simples e fácil de entender
	•	Código organizado para facilitar manutenção e aprendizado
Como Utilizar
	1.	Execute o programa
	2.	Informe os números desejados
	3.	Escolha a operação matemática
	4.	O resultado será exibido na tela
Observações
	•	O projeto foi desenvolvido com foco em simplicidade
	•	Não possui validações avançadas
	•	Ideal para iniciantes que desejam entender a lógica de uma calculadora.
